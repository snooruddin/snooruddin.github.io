1. Return Largest number in arrays

So we Have an array that contains more arrays. We have to find the maximum number in each subarray and create a array of all the maximum numbers.

This is just a test of nested loops. 

                

                    function largestOfFour(arr) {

                        // array containing the answer
                        var ans = [];

                        // this loop traverses the given array
                        for(var i = 0; i < arr.length; i++)
                        {

                            //this holds the maximum for each subarray
                            var maximum = 0;

                            //this loop traverses each subarray of the given array
                            for(var j = 0; j < arr[i].length; j++)
                            {

                                //updating maximum
                                if(arr[i][j] > maximum )
                                {
                                    maximum = arr[i][j];
                                }
                            }

                            //pushing the maximum of each subarray into the ans array
                            ans.push(maximum);
                        }
                        return ans;
                    }

                    largestOfFour([[4, 5, 1, 3], [13, 27, 18, 26], [32, 35, 37, 39], [1000, 1001, 857, 1]]);

2. Confirm the ending

In this case, we have to find weather a given string contains another given string at the ens

We cannot use endsWith() as fcc wants us not to. 

                

                    function confirmEnding(str, target) {

                        // substr(startIndex, endIndex) is a built-in method that returns the string[startIndex] to not including string[endIndex] in a string.
                        // "Hello".substr(3, 5) returns "lo" as "Hello"[3] = lo ans "Hello"[5] = ""
                        // one thing to notice is that string[endIndex] is not included in the resultant string
                        // substr(startIndex) by defaults returns from string[startIndex] to string[string.length]
                        var end = str.substr(str.length - target.length);

                        return end === target;
                    }

                    confirmEnding("Bastian", "n");

We can also use the endsWith() built-in method 

                
                    function func(str, end)
                    {
                        return str.endsWith(end);
                    }
                    func("bonjour", "ur");

3. Repeat a string repeat a string

This is the easiest one. You just run a for loop and add the given string with an empty string. 

                

                    function repeatStringNumTimes(str, num) {

                        // ans contains the resultant string
                        var ans = "";

                        // add the given string with the empty string the given number of times
                        for(var i = 0; i < num; i++)
                        {
                            ans += str;
                        }
                        return ans;
                    }

                    repeatStringNumTimes("abc", 3);

4. Truncate a string

In this case, we have to truncate the additional parts of a string that exceeds the given length and then add some characters with the string.

This is just a case of writing the if-else statements. We have three cases.

Case 1: if (string.length < num ) we dont have to truncate anything

Case 2: if (string.length > num && num > 3) we have to truncate from index 0 to index num - 3 for adding the "..." string.

Case 3: if (string.length > num && num <= 3 ) we have to truncate from index 0 to index num 

                

                    function truncateString(str, num) {

                        //case 2
                        if(str.length > num && num > 3)
                        {

                            //slice till num - 3 because we have to think about length of "..." in the resultant string
                            str = str.slice(0, num - 3 ) + "...";
                            return str;
                        }

                        //case 3
                        else if(str.length > num && num <= 3)
                        {

                            // as num <= 3, we don't have to care about the length of "..." in the resultant string
                            str = str.slice(0, num) + "...";
                            return str;
                        }

                        //case 1
                        else
                        {
                            return str;
                        }
                    }

                    truncateString("A-tisket a-tasket A green and yellow basket", 11);

5. Chunky Monkey

This is another case of simple for loop that increments with size.

Basically, we have to slice the array into groups of arrays and push the slices into a resultant array. 

                

                    function chunkArrayInGroups(arr, size) {

                        // contains the resultant array
                        var res = [];

                        // notice the i+=size
                        // so if slice is 4, the first slice will be 0 to 3 , second will be 4 to 7 and so on
                        for(var i = 0; i < arr.length; i+= size)
                        {
                            // pushing the slices into the resultant array
                            res.push(arr.slice(i, i+size));
                        }
                        return res;
                    }

                    chunkArrayInGroups(["a", "b", "c", "d"], 2);
