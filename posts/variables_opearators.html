Variables

Javascript has three keywords to declare variables: let, var, const. 

                
                    const pi = 3.1416;
                    //const is for assigning values to variables that wont change the initial value. For example pi is a constant.

let uses block scope, meaning a variables scope is only on the function you declared the variable in.

                
                    let x = 1;
                    if (x === 1) {
                      let x = 2;
                      console.log(x);
                      // x = 2 only for this function block
                      // output: 2
                    }

                    console.log(x);
                    // output: 1

var creates a global variable as opposed to let 

                
                    var x = 1;
                    if (x === 1) {
                      var x = 2;
                      console.log(x);
                      // expected output: 2
                    }

                    console.log(x);
                    // see how var redeclaration has changed the initial value of x
                    // expected output: 2

Javascript doesn't need implicitly declaring data types of variables. Meaning you can set any type of data to a variable. 

                
                    var i; //i is "undefined' at this point
                    i = 5;
                    i = "Hello World!";
                    i = true;
                    //you can also set objects and arrays to variables but they will be discussed in future topics

Operators

Javascript supports operators +, -, *, /, %, **, +=, -=, =, ==, ===, !=, !== 

                
                    //adding two numbers
                    var i = 3 + 4; //i = 5
                    var i = 4.567655;
                    //adding two variables
                    var j = 1, k = 2;
                    var i = j + k;
                    //adding strings or string concatenation
                    var i = "hello" + "World! ";
                    var i = 3 + "i"; //i = "3i";
                    var i = "i" + 3; // i = "i3";

                    //subtract numbers
                    var i = 5 - 3; //i = 2
                    //subtraction doesn't work with strings

                    //multiplying numbers
                    var i = 5 * 3;

                    //dividing numbers
                    var i = 4 / 3;

                    //% means "remainder" in javascript, not "modulo"
                    //% takes the sign of the dividend unlike modulo
                    12 % 5 // 2
                    -1 % 2 // -1
                    NaN % 2 // NaN

                    // " ** " is the exponentiation operator
                    var i = 2**3 // i = 8;

                    //+= augmented plus
                    var i = 3;
                    i += 5; //equals to i = i + 5;

                    //-= augmented subtract
                    var i = 3;
                    i -= 2; //equals to i = i - 2;

                    //= assignment operator
                    var i = 3; // setting i to 3;

                    //== checks weather right side is equal to left side and returns boolean
                    //doesn't care about type
                    2 == 3 //false
                    2 == '2' //true
                    'hello' == 'hello' //true

                    //=== is assignment operator that checks the type of the sides also
                    3 === '3' //false

                    //!= checks right side with left side and return boolean\
                    // doesn't match types
                    2 != '2' //false
                    2 != 3 //true

                    //!== checks the type of both sides
                    2 !== '2' //true
                    2 !== 2 //false
