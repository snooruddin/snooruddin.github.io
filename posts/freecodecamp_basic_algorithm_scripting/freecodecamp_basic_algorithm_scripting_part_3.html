1. Slasher Flick

In this case, we have to chop elements from the given array.

splice() is a built in function that removes elements from array. 

                

                    function slasher(arr, howMany) {

                        //splice(startIndex, howManyToSplit) starts from the startIndex and separates as many indicated by the howManyToSplit
                        // One thing to notice is that splice changes the contents of the array it works on
                        // splice can be used to both remove and add data to an array
                        arr.splice(0, howMany);
                        return arr;
                    }

                    slasher([1, 2, 3], 2);

Another way is to use slice(). slice(startIndex, endIndex) slices an array from the given startIndex to the endIndex and returns and array containing the sliced elements.

If no endIndex is given, slice() slices till end Of Array

slice() doesn't change the contents of the working array.

                

                    function slasher(arr, howMany) {

                        // slice stats from the index specified by the howMany argument and slices to the end Of array
                        arr = arr.slice(howMany);
                        return arr;
                    }

                    slasher([1, 2, 3], 2);

2. Mutations

In this case, we have to find weather all the characters in the second argument string is present in the first argument string.

                

                    function mutation(arr) {

                        // its easy to compare the strings when we don't have to worry about weather the character is uppercase or lowercase
                        // indexOf(something) returns the index of the first occurrence of "something" in the string its working on
                        // indexOf() returns -1 if no match is found
                        arr[0] = arr[0].toLowerCase();
                        arr[1] = arr[1].toLowerCase();

                        // traverses all the characters of the first string
                        for(var i = 0; i < arr[1].length; i++)
                        {
                            // if one character is not present, return false
                            if(arr[0].indexOf(arr[1][i]) == -1)
                            {
                                return false;
                            }
                        }

                        // else every item was present in the first string
                        return true;
                    }

                    mutation(["hello", "hey"]);

3. Falsy Bouncer

In this case, we have to delete all falsy values from an array. Falsy values are 0, null, undefined , false, NaN and "". 

                
                    function bouncer(arr) {

                        // filter() uses a callback function to determine weather a property is true or false for each element of the array
                        // the boolean object returns false for every falsy elements in the array and thus the falsy elements are cut out.
                        // filter() doesn't change the contents of the array it is works on
                        arr = arr.filter(Boolean);

                        return arr;
                    }

                    bouncer([7, "ate", "", false, 9]);

4. Seek And Destroy

In this case, we have to seek for the given elements in the arr array and delete them from the arr if found. 

                
                    function destroyer(arr) {

                        // res will contain the resultant array
                        var res = [];

                        var isDuplicate = false;

                        // traverses each element of the array
                        for(var i = 0; i < arr.length; i++)
                        {

                            isDuplicate = false;

                            // traversing all the arguments
                            // arguments is an array like object that contains all the arguments passed to a function....
                            // arguments[0] contains the first argument passed to a function
                            // arguments[1] contains the second argument passed to a function and so on and so forth...
                            for(var j = 1; j < arguments.length; j++)
                            {
                                // if the current element of the arr that is being traversed equals to the current argument,
                                // we set the isDuplicate to true , so that we don't have to insert it into the res array
                                if(arr[i] === arguments[j])
                                {
                                    isDuplicate = true;
                                    break;
                                }
                            }

                            // if the item in the array is not duplicate, it is inserted into the res array
                            if(isDuplicate == false)
                            {
                                res.push(arr[i]);
                            }
                        }
                        return res;
                    }

                    destroyer([1, 2, 3, 1, 2, 3], 2, 3);

5. Where Do I Belong?

Ah... The question of every single human being on earth...

This one is about sorting and finding one's place .

                

                    function getIndexToIns(arr, num) {

                        // sort() uses a callback function to sort the item it works on.
                        // if puts a before b if the function returns negative results
                        // it puts b before a if the function returns positive results
                        // it puts a before b if the function returns 0

                        //in this case we want to sort an array in ascending order.
                        // The following comparator function returns positive if a > b,then the sort() function puts b before a
                        // returns negative if b > a, so sort() puts a before b
                        function ascendingSortHelper(a,b)
                        {
                            return a - b;
                        }

                        // adding the new number to array
                        arr.push(num);

                        // sorting using the comparator function
                        // sort() changes the array it works on
                        arr.sort(ascendingSortHelper);

                        // returning the index of the first occurrence of the num
                        return arr.indexOf(num);
                    }

                    getIndexToIns([40, 60], 50);

6. Caesars Cipher

Caesars cipher is basically a rotating cipher. Meaning, Case 1: if the ASCII CODE of the character is less than 78 you have to add 13 with the ASCII CODE to decipher.

Else ,Case 2: you just have to subtract 13 from the character ASCII CODE. 

                

                    function rot13(str) {

                        // will contain the resultant string
                        var res = "";   // LBH QVQ VG!

                        // traverses the string
                        for(var i = 0; i < str.length; i++)
                        {
                            // charCodeAt() returns the ASCII CODE of a character
                            // The problem specified that all the string will be uppercase
                            if(str.charCodeAt(i) >= 65 && str.charCodeAt(i) <= 90 )
                            {
                                //Case 1
                                if(str.charCodeAt(i) < 78)
                                {

                                    // fromCharCode() builds a string from the given ASCII CODE
                                    // It operates on the Built-in String Object
                                    res += String.fromCharCode( str.charCodeAt(i) + 13);
                                }

                                //Case 2
                                else
                                {
                                    res += String.fromCharCode( str.charCodeAt(i) - 13);
                                }
                            }
                            //If other symbols, ignore
                            else
                            {
                                res += str[i];
                            }
                        }
                        return res;
                    }
