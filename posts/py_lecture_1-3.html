Objects & Numerical types

Objects are the core things of python. Every object has a type that defines the set of things that can be done with the instance of that object.

Types are either scalar or non-scalar. Scalar objects are indivisible. Python has four types of scalar objects:

  • int represents integer. E.g. 50, -40 etc.
  • bool represents Boolean values True and False
  • float represents real numbers e.g. 4.56, -3.1414 etc.
  • None represents the absence of a value

Strings are an example of a non-scalar object.

Branching and Indentation

Python forces the user to write well indented code. Unlike C/C++, python has no "code block" per se, meaning, you can't create a code block using {}.

A code block in python is created using colon(:) after a statement and writing the statements of the code block with same level of indentation.

                        
                            #a program to find maximum from three input numbers

                            #getting the inputs from user and typecasting them to floats
                            a = float(input("Enter a: "))
                            b = float(input("Enter b: "))
                            c = float(input("Enter c: "))

                            #maximum will hold the maximum number in the end
                            maximum = 0;

                            #so, here a code block begins
                            if a > b:
                                if b > c: #if the next statement is indented to four spaces, it means the statement is part of the codeblock
                                    maximum = a
                                else:
                                    if a > c:
                                        maximum = a
                                    else:
                                        maximum = c
                            else:
                                if a > c:
                                    maximum = b
                                else:
                                    if b > c:
                                        maximum = b
                                    else:
                                        maximum = c
                            print("maximum:",maximum)

So if you want a statement to be a part of a block, you have to indent the statement

Strings

Python strings are immutable meaning the unlike C/C++, you can't change the individual characters of the string once the string has been initialized. 

                        
                            string = "I am Spartaa!!!"

                            #individual characters of the string can be accessed by indexing
                            print(string[5]) #Output: S

                            #negatice indice means last char in python
                            print("i am sparta"[-1]) #Output: a
                            print("i am sparta"[-2]) #Output: t

                            string[5] = "r" #this generates an error and strings are immutable

                            string = "pew pew pew" #this is totally OK, as you are just re-binding the string

                            #using len() to find length of string
                            print(len(string)) #Output: 11

                            #slicing is used to extract sbstrings of specific lengths from a string
                            string[0:len(string)] #Output: pew pew pew

                            #slice(start, stop, step)
                            #not gonna include string[stop]
                            #bydefault step = 1
                            #colon followed by nothing means until end or until beginning
                            print("i am sparta"[3:]) #Output: m sparta

                            s = "i am sparta"
                            print(s[:len(s)]) #Output: i am sparta
                            print(s[:len(s) - 1]) #Output: i am spart

                            print("i am sparta"[:-1]) #Output: i am spart

                            s = "i am sparta"
                            print(s[5: 8: 2]) #Output: sa
                            print(s[10: 5: -2]) #Output: arp
                            print(s[::]) #Output: i am sparta
                            print(s[::-1]) #prints the string reversely
