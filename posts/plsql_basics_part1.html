PL/SQL

I use Oracle DBMS 11g express edition. In Oracle DBMS PL/SQL is a procedural language extension to SQL. It is different from SQL, but the basics are same.


Creating a user and Granting permission to do stuff

We have to run SQL commands from the "Run SQL commands" terminal. You can find it in Start Menu of Windows.

At first we connect to the system account, this is a built in administrator account meaning it can create users, delete users and much more. 

                
                    connect system;
                    -- you create a comment in sql using the dash operator twice
                    -- this is a comment in sql

This then requires a password. The password is the password you used when setting up the database software.
