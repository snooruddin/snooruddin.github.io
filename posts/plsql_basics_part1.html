PL/SQL

I use Oracle DBMS 11g express edition. In Oracle DBMS PL/SQL is a procedural language extension to SQL. It is different from SQL, but the basics are same.


Creating a user and Granting permission to do stuff

We have to run SQL commands from the "Run SQL commands" terminal. You can find it in Start Menu of Windows.

At first we connect to the system account, this is a built in administrator account meaning it can create users, delete users and much more. 

                
                    CONNECT system;
                    -- sql statements end with a semicolon
                    -- you create a comment in sql using the dash operator twice
                    -- this is a comment in sql

This then requires a password. The password is the password you used when setting up the database software. The password is not echoed meaning you cant see the password as you type it. So type carefully.

After successful login a user is created like this 

                
                    CREATE USER put_username_here IDENTIFIED BY put_password_here;

After successful user creation you need to grant permission to the user so that he can create table and do other stuff. This can be done in two ways. You can either Grant him permissions one by one or you can grant all permission to him at once. Here is a link to all the permissions you can permit to a user. To Give a user all permissions at once: 

                
                    GRANT ALL PRIVILEGES TO username_here;

Now that a user with all the permissions have been created, you can disconnect from system and login to the new user. 

                
                    DISCONNECT;    -- to disconnect
                    CONNECT username;    --to login to new user

You have to put the password for the username.
